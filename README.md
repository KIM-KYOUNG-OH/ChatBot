## 프로젝트 목표
- 전국 산불 및 화재 정보를 한 눈에 볼 수 있는 실사용 가능한 알림 서비스 배포
- 단순한 기능 구현 뿐 아니라 대용량 트래픽 처리까지 고려한 기능 구현
- 서비스 구현 과정에서 지속적인 리펙토링을 통한 가독성이 좋고 유지보수에 용이한 코드 작성
